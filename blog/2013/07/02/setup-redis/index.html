安装 Redis:一步至简

Jul 2nd, 2013

Redis 的安装超级简单，对于任何级别的 Linux 用户都没有障碍。

只需简单几步：

  1. 下载源码。

wget http://redis.googlecode.com/files/redis-2.6.14.tar.gz

  1. 解压编译。

$ tar vzxf redis-2.6.14.tar.gz $ cd redis-2.6.14 $ make

然后直接启动编译好的二进制 server 端：

$ src/redis-server

这时打开客户端即可尝试一下 Redis 的语法了：

$ src/redis-cli redis> set foo bar OK redis> get foo “bar”

原文：http://redis.io/download
